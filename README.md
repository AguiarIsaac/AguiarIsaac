# Olá! Eu sou o Isaac Aguiar 👋

Backend Developer | Software Engineering | Java | Integrações | Sankhya ERP

Sou desenvolvedor de software com foco em construção de aplicações back-end, integrações entre sistemas e arquitetura de soluções corporativas.

Atualmente desenvolvo aplicações utilizando **Java/Spring Boot**, **React**, **Angular** e **Kotlin**, criando APIs, dashboards, aplicações Android TV e soluções integradas ao ecossistema **Sankhya ERP**.

Tenho experiência com:

* 🚀 Desenvolvimento de APIs REST e integrações corporativas
* 🏗️ Arquitetura de aplicações e documentação técnica (ADRs)
* 🔗 Integração entre Oracle, PostgreSQL e serviços externos
* 📦 Docker, containers e organização de ambientes
* ⚙️ Azure DevOps e versionamento de código
* 📊 Desenvolvimento de aplicações para apoio operacional e automação de processos

Atualmente estou aprofundando meus estudos em:

* 📚 Pós-graduação em Engenharia de Software
* ☕ Arquitetura Backend

* ☁️ Containers, DevOps e escalabilidade
